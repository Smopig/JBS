1. 環境準備

需要的檔案

wafer_tool/
├── index.html            ← 主程式（本頁）
├── 操作手冊.md
└── libs/
    ├── plotly.min.js     ← 圖表庫
    └── xlsx.full.min.js  ← Excel 讀寫庫
若 libs/ 內缺少檔案，開啟工具時會出現紅色錯誤。請先在有網路的環境下開啟 download_libs.html（只需執行一次）下載後即可完全離線使用。

開啟方式

用任意瀏覽器（Chrome / Edge / Firefox）直接開啟 index.html，無需伺服器。所有資料均在本機處理，不上傳任何伺服器。

2. 📂 資料載入

上傳檔案

  • 點擊上傳區，或將 Excel 檔案拖曳進去。
  • 支援 .xls / .xlsx，可一次選取多個檔案（每個檔案 = 一個 Lot）。
  • Lot 名稱自動從檔名解析：取第一個 - 之後的部分。
    例：CP-L24A001.xlsx → Lot = L24A001

設定列號

欄位說明預設值
標題列號Excel 中欄位名稱所在的列（從 1 開始）13
資料起始列號實際量測資料從哪一列開始30

設定好後點擊 「① 載入欄位」。

欄位命名

載入欄位後，下方出現欄位命名表格。「參考數值」欄顯示該欄位第 14～19 列的樣本值，輔助判斷量測條件。

名稱前綴自動換算用途
IR_原始值 × 1,000,000（A → μA）漏電流
VR_四捨五入至小數 1 位逆向崩潰電壓
VF_四捨五入至小數 3 位順向電壓
其他不換算—

🔄 自動命名：根據原始欄位名稱中的關鍵字（VFBC / BVCBO / ICBO）及樣本值解析條件，自動填入標準名稱（如 VF_5mA、VR_10uA、IR_1V）。

確認命名後點擊 「③ 套用命名並開始分析」，工具自動讀取所有檔案、套用換算、啟用所有分析分頁。

6. 📋 統計

量測項目選擇

頂部可新增多個下拉選單，每個選單對應一個量測項目，各自產生獨立統計區塊。

統計列開關（Chip）

預設顯示：Count Min Max Mean Median Std

  • 點擊 Chip 的 × 從所有項目隱藏；點擊 + 重新加入。
  • 每列右側的 − 按鈕：只對該項目隱藏這一列（不影響其他項目）。
  • 區塊標題列的「↩ 還原隱藏列」可復原該項目所有隱藏列。

SPEC 設定

欄位說明
LSL下規格限；數值低於此值時格子標紅
Typical目標典型值（僅供參考，不影響顏色）
USL上規格限；數值高於此值時格子標紅

數值格式

選項說明
0.0（固定小數）預設；Count 為 0 位，其餘 2 位
1e（科學符號）適合極小或極大數值
位數小數位數（0～10）

自訂計算

輸入名稱和公式，可用的變數：

Count Min Max Mean Median Std LSL USL Typical

公式範例說明
Math.min((USL-Mean)/(3*Std),(Mean-LSL)/(3*Std))Cpk
(USL-LSL)/(6*Std)Cp
Max-MinRange
(Mean-LSL)/(3*Std)Cpl
(USL-Mean)/(3*Std)Cpu

7. 📉 圖表分析

通用控制

控制說明
量測項目可新增多個，同時分析多個參數
Lot 顯示全部疊加：所有 Lot 在同一張圖（不同色）
每 Lot 獨立：每 Lot 各自一張圖
每列欄數1 / 2 / 3 / 4 / 6 或自訂
SPEC 線勾選後顯示 LSL（紅虛線）/ Typical（綠點線）/ USL（紅虛線）水平線

圖表類型

📦 箱型圖（Box Plot）

X 軸 = 各 Lot，Y 軸 = 量測值。顯示中位數、四分位距、最大最小值、離群值。每個量測項目一張圖。

🎻 小提琴圖（Violin Plot）

同箱型圖佈局，額外顯示資料的機率密度形狀，內含 Box 及均值線。

🔵 散佈圖（Scatter Plot）

  • X 軸：Xpos / Ypos / Index
  • 疊加模式：所有 Lot 同圖（不同色）；獨立模式：每 Lot 一張
  • 點大小滑桿可調整

📉 趨勢圖（Trend）

  • 以 Lot 為 X 軸，Y 軸為每個 Lot 的均值 ± 誤差棒
  • 誤差棒：±1σ / ±2σ / ±3σ / Min～Max
  • Cpk 標示：需設定 LSL 與 USL，勾選後在各點旁標示 Cpk 值
  • 獨立模式下顯示逐點數值折線（含均值參考線）

📊 長條圖 + 誤差棒（Bar）

X 軸 = Lot，長條高度 = 均值，誤差棒選項與 Cpk 標示同趨勢圖。每個量測項目一張圖。

🎯 SPC

  • 顯示每個量測點，並繪製 CL（中心線）/ UCL / LCL
  • X 軸：Index / Lot 分組 / Xpos / Ypos
  • 管制線 σ：±1σ / ±2σ / ±3σ（預設 ±3σ）
  • 超出管制線的點以紅色標示，圖表標題顯示 CL / UCL / LCL 數值

📈 CDF（累積分布）

  • X 軸 = 量測值，Y 軸 = 累積百分比（0～100%）
  • 疊加模式 = 所有 Lot 同圖；獨立模式 = 每 Lot 一張（含填色）

🔗 相關分析（Correlation）

  • X 軸 = 量測項目選單的項目；Y 軸 = 獨立下拉選單的第二個項目
  • 各 Lot 以不同顏色區分；可用 Lot 篩選選擇全部或個別 Lot
  • 迴歸線 + R²：勾選後疊加最小平方迴歸線，圖角顯示方程式與 R²
  • 依 SPEC 篩選資料：勾選後，只顯示並計算 X、Y 皆在各自 SPEC 範圍內的點；迴歸標示加上「[SPEC 內]」備註
  • Y 軸的 SPEC 在 SPEC 列以紫色 Y:標示獨立輸入

9. 常見問題

❓ 開啟後出現紅色「找不到必要的 Library 檔案」

libs/ 資料夾內缺少 plotly.min.js 或 xlsx.full.min.js。請在有網路的環境下開啟 download_libs.html 下載後放入 libs/，之後即可完全離線使用。

❓ 上傳後 Lot 名稱顯示不正確

Lot 名稱取第一個 - 之後的部分。若檔名不含 -，則以完整去除副檔名的檔名作為 Lot 名稱。

❓ 資料列號設定錯誤

修改「標題列號」與「資料起始列號」後，重新點擊「① 載入欄位」再執行「③ 套用命名並開始分析」即可。

❓ Cpk 計算結果為空白

Cpk 需要 LSL 和 USL 都有值才能計算。請在統計分頁或圖表分析的 SPEC 輸入欄中填入 LSL 與 USL。

❓ 相關分析沒有出現迴歸線

請確認已在「Y 軸項目」下拉選單選擇了一個與 X 軸不同的量測項目，且「迴歸線 + R²」勾選框已打勾。

❓ SPC 圖的紅色點是什麼

紅色點表示該量測值超出 UCL（上管制線）或低於 LCL（下管制線），屬於失控點。
