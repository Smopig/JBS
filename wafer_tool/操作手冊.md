## 1. 環境準備

### 1.1 需要的檔案

```
wafer_tool/
├── index.html          ← 主程式
├── 操作手冊.md          ← 本手冊
└── libs/
    ├── plotly.min.js   ← 圖表庫
    └── xlsx.full.min.js← Excel 讀寫庫
```

若 `libs/` 內缺少檔案，開啟工具時會出現紅色錯誤提示。請先開啟 `download_libs.html`（需要網路，只需執行一次）下載後即可離線使用。

### 1.2 開啟方式

用任意瀏覽器（Chrome / Edge / Firefox）直接開啟 `index.html`。

## 2. 📂 資料載入 分頁

### 2.1 上傳檔案

- 點擊上傳區或將 Excel 檔案**拖曳**進去。
- 支援 `.xls` / `.xlsx`，可**一次選取多個檔案**（每個檔案對應一個 Lot）。
- Lot 名稱自動從檔名解析：檔名格式 `前綴-LotName.xlsx`，取第一個 `-` 之後的部分為 Lot 名稱。

### 2.2 設定列號

| 欄位 | 說明 | 預設值 |
|------|------|--------|
| 標題列號 | Excel 中欄位名稱所在的列（從 1 開始計算） | 13 |
| 資料起始列號 | 實際量測資料從哪一列開始 | 30 |

設定好後點擊 **「① 載入欄位」**。

### 2.3 欄位命名

載入欄位後，下方會出現欄位命名表格：

| 欄 | 說明 |
|----|------|
| 原始欄位名稱 | Excel 中的原始欄位 |
| 新名稱 | 可直接修改，分析時使用此名稱 |
| 數值換算方式 | 依名稱前綴自動判斷（見下表） |
| 參考數值 | 該欄位第 14～19 列的樣本值，輔助判斷量測條件 |

**命名前綴換算規則：**

| 前綴 | 換算 | 用途 |
|------|------|------|
| `IR_` | 原始值 × 1,000,000（A → μA） | 漏電流 |
| `VR_` | 四捨五入至小數 1 位 | 逆向崩潰電壓 |
| `VF_` | 四捨五入至小數 3 位 | 順向電壓 |
| 其他 | 不換算 | — |

**自動命名按鈕**：根據原始欄位名稱中的關鍵字（VFBC / BVCBO / ICBO）及樣本值解析條件標籤，自動填入標準名稱（如 `VF_5mA`、`VR_10uA`、`IR_1V`）。

### 2.4 套用並分析

確認命名後點擊 **「③ 套用命名並開始分析」**，工具自動：
1. 讀取所有檔案並合併資料
2. 套用換算規則
3. 解析 Lot 清單與量測項目
4. 啟用所有分析分頁

## 3. 🗺 Wafer MAP 分頁

每個 Lot 獨立一張熱圖，以 (Xpos, Ypos) 坐標顯示量測值。

### 3.1 控制選項（第一列）

| 選項 | 說明 |
|------|------|
| 量測項目 | 選擇要顯示哪個量測參數 |
| 每列欄數 | 1 / 2 / 3 / 4 / 6 或「自訂…」 |
| X 軸方向 | 大→小（預設，X 大值在左）或 小→大 |
| Y 軸方向 | 大→小（預設，Y 大值在上）或 小→大 |

### 3.2 控制選項（第二列）

| 選項 | 說明 |
|------|------|
| 色階樣式 | 白→紅 / 藍↔紅 / Viridis / 彩虹 / 黃→橙→紅 / 灰階 |
| 色階範圍 | 手動輸入 min / max；留白則自動計算 |
| 自動填入 | 根據目前項目的全域最小/最大值填入 |
| 清除 | 清空手動輸入，恢復自動 |

## 4. 📊 Count 分布 分頁

顯示每個 Lot 的量測值次數分布直方圖，疊加常態分布曲線。

### 控制選項

| 選項 | 說明 |
|------|------|
| Lot 篩選 | 全部 / 指定單一 Lot |
| 量測項目 | 選擇量測參數 |
| 每列欄數 | 1 / 2 / 3 / 4 / 6 或自訂 |
| X 軸範圍 | 手動輸入 min～max；留白自動 |
| Y 軸範圍 | 手動輸入 min～max；留白自動 |
| 還原預設 | 清除所有手動輸入 |

圖表標題顯示 n（樣本數）、mean、std。

---

## 5. 📈 % 分布 分頁

與 Count 分布類似，Y 軸以百分比（%）顯示，所有 Lot 使用**相同的 X 軸 bin 邊界**（全域統一），方便 Lot 間比較。

### 控制選項

| 選項 | 說明 |
|------|------|
| Lot 篩選 | 全部 / 指定單一 Lot |
| 量測項目 | 全部項目 / 指定單一項目 |
| 每列欄數 | 1 / 2 / 3 / 4 / 6 或自訂 |
| X 軸範圍 | 手動輸入；留白自動（使用全域 bin 範圍） |
| Y 軸範圍 | 預設 0～100%；可手動縮放 |

## 6. 📋 統計 分頁

以表格形式顯示每個量測項目對各 Lot 的統計數值。

### 6.1 量測項目選擇

- 頂部可新增多個下拉選單，每個選單選一個量測項目。
- 每個項目產生一個獨立統計區塊，可捲動瀏覽。
- 點擊「＋ 新增項目」可同時比較多個參數。

### 6.2 統計列控制

**顯示統計列（Chip 開關）**：

- 預設顯示：Count / Min / Max / Mean / Median / Std
- 點擊 Chip 上的 `×` 從所有項目隱藏該統計列；點擊 `+` 重新加入。
- 「↩ 一鍵還原預設」恢復全部預設統計列。

**每個項目的個別隱藏**：

- 每一統計列右側有 `−` 按鈕，只對該項目隱藏這一列。
- 項目區塊標題列會出現「↩ 還原隱藏列（N）」按鈕可一鍵復原。

### 6.3 SPEC 設定（每個統計列 × 每個項目）

每行右側有三個輸入框：

| 欄 | 說明 |
|----|------|
| LSL | 下規格限；數值低於此值時，對應格子標紅 |
| Typical | 目標典型值（不影響顏色，僅顯示參考線） |
| USL | 上規格限；數值高於此值時，對應格子標紅 |

### 6.4 數值格式

每一統計列右側可設定：

| 欄 | 說明 |
|----|------|
| 格式 | `0.0`（固定小數）或 `1e`（科學符號） |
| 位數 | 小數位數（0～10） |

預設：Count 為 0 位；其餘統計項目為 2 位固定小數。

### 6.5 自訂計算

在「自訂計算」列輸入：

- **名稱**：自訂統計列的標籤（如 `Cpk`）
- **公式**：JavaScript 運算式，可用變數：
  `Count` `Min` `Max` `Mean` `Median` `Std` `LSL` `USL` `Typical`

**範例：**

| 公式 | 說明 |
|------|------|
| `(USL - LSL) / (6 * Std)` | Cp |
| `Math.min((USL - Mean) / (3 * Std), (Mean - LSL) / (3 * Std))` | Cpk |
| `Max - Min` | Range |
| `(Mean - LSL) / (3 * Std)` | Cpl |

點擊「新增」後，自訂列會出現在 Chip 清單，可正常開關與設定 SPEC/格式。

## 7. 📉 圖表分析 分頁

提供 8 種圖表類型，支援多項目比較、SPEC 線疊加與 Lot 顯示模式切換。

### 7.1 通用控制

| 控制 | 說明 |
|------|------|
| 量測項目 | 可新增多個下拉選單，同時分析多個參數 |
| Lot 顯示 | **全部疊加**：所有 Lot 在同一張圖；**每 Lot 獨立**：每個 Lot 各自一張圖 |
| 每列欄數 | 1 / 2 / 3 / 4 / 6 或自訂 |
| SPEC 線 | 勾選後顯示 LSL（紅虛線）/ Typical（綠點線）/ USL（紅虛線）的水平線 |
| SPEC 輸入 | 每個量測項目下方可分別輸入 LSL / Typical / USL |

### 7.2 圖表類型說明

#### 📦 箱型圖（Box Plot）
- X 軸 = 各 Lot；Y 軸 = 量測值
- 顯示中位數、四分位距、最大最小值、離群值
- 每個量測項目一張圖，Lot 間自動分組

#### 🎻 小提琴圖（Violin Plot）
- 同箱型圖佈局，額外顯示資料的機率密度形狀
- 內含 Box 及均值線

#### 🔵 散佈圖（Scatter Plot）
- **X 軸**可選：Xpos / Ypos / Index
- **Lot 顯示**：疊加 = 所有 Lot 同圖（不同色）；獨立 = 每 Lot 一張圖
- **點大小**滑桿可調整

#### 📉 趨勢圖（Trend）
- 以 Lot 為 X 軸，顯示每個 Lot 的**均值 ± 誤差棒**
- **誤差棒**可選：±1σ / ±2σ / ±3σ / Min～Max
- **Cpk 標示**：勾選後在各 Lot 均值點上方顯示 Cpk 值（需設定 LSL / USL）
- 疊加模式 = 一張圖；獨立模式 = 每 Lot 一張圖（顯示逐點數值折線）

#### 📊 長條圖 + 誤差棒（Bar + Error）
- X 軸 = Lot，長條高度 = 均值，誤差棒 = ±Nσ 或 Range
- 誤差棒與 Cpk 標示選項同趨勢圖
- 每個量測項目一張圖

#### 🎯 SPC
- 顯示每個量測點（單點），並繪製管制線 CL / UCL / LCL
- **X 軸**可選：Index / Lot 分組 / Xpos / Ypos
- **管制線 σ**：±1σ / ±2σ / ±3σ（預設 ±3σ）
- 超出管制線的點以**紅色**標示
- 圖表標題顯示 CL、UCL、LCL 數值

#### 📈 CDF（累積分布）
- X 軸 = 量測值；Y 軸 = 累積百分比（0～100%）
- 疊加模式 = 所有 Lot 在同一張圖；獨立模式 = 每 Lot 一張圖（含填色）
- 適合快速比較各 Lot 的分布位移

#### 🔗 相關分析（Correlation）
- X 軸 = 所選量測項目；Y 軸 = 獨立下拉選單選定的第二個項目
- 各 Lot 以不同顏色區分
- **Lot 篩選**：可選全部 Lot 或個別 Lot
- **迴歸線 + R²**：勾選後疊加最小平方迴歸線，並在圖角顯示方程式和 R² 值
- **依 SPEC 篩選資料**：勾選後，只顯示 X 和 Y 皆在各自 SPEC 範圍內的點，迴歸線與 R² 也只由這些點計算；迴歸標示加上「[SPEC 內]」備註
- Y 軸項目的 SPEC（LSL / Typical / USL）在 SPEC 欄位列以紫色「Y:」標示單獨輸入

## 8. 💾 匯出 Excel

點擊任一頁面右上角的「💾 匯出 Excel」，會產生含兩個工作表的 Excel 檔：

| 工作表 | 內容 |
|--------|------|
| 合併資料 | 所有 Lot 的原始量測資料（已套用換算）合併為單一表格 |
| 統計 | 每個量測項目 × 每個 Lot 的 Count / Min / Max / Mean / Median / Std |

檔名格式：`wafer_analysis_YYYY-MM-DD.xlsx`

## 9. 常見問題

**Q：開啟後出現紅色「找不到必要的 Library 檔案」錯誤？**
A：`libs/` 資料夾內缺少 `plotly.min.js` 或 `xlsx.full.min.js`。請在有網路的環境下開啟 `download_libs.html` 下載後放入 `libs/`，之後即可完全離線使用。

---

**Q：上傳後 Lot 名稱顯示不正確？**
A：Lot 名稱從檔名解析，規則是取第一個 `-` 之後的部分。  
例如：`CP-L24A001.xlsx` → Lot = `L24A001`  
若檔名不含 `-`，則以完整去除副檔名的檔名作為 Lot 名稱。

---

**Q：資料列號設定錯誤怎麼辦？**
A：修改「標題列號」與「資料起始列號」後，重新點擊「① 載入欄位」再執行「③ 套用命名並開始分析」即可。

---

**Q：Cpk 計算結果為空白？**
A：Cpk 需要 LSL 和 USL 都有值才能計算。請在統計分頁或圖表分析的 SPEC 輸入欄中填入 LSL 與 USL。

---

**Q：相關分析圖沒有出現迴歸線？**
A：請確認已在「Y 軸項目」下拉選單選擇了一個與 X 軸不同的量測項目，且「迴歸線 + R²」勾選框已打勾。

---

**Q：SPC 圖的顏色怎麼判讀？**
A：各 Lot 的點以不同顏色區分；**紅色點**表示該點超出 UCL 或低於 LCL（管制線外）。
